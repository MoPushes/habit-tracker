Description:
A simple web application to help build
and maintain good habits while keeping track
on a daily basis. Monitor your progress daily,
stay consistent and visualize your streaks
over time.

Setup:
More details to be added after update of the
Tech Stack.

Usage:
- Add a new habit using the input field
- Click on a habit to mark it as complete for the day
- Track your streak and stay consistent
- Edit or delete habits anytime
